Controller的使用

@controller 处理http请求

@RestController Spring4后新加的注解， 原来返回json需要@ResponseBody配合@Controller

@RequestMapping 设置url映射。

@RequestMapping(“path”)指定url路径

@PathVariable 获取url中的数据

@RequestParam 获取请求参数的值，传统的url?key=value这样的url链接

@GetMapping 组合注解，相当于把@RequestMapping中的method设置为RequestMethod.GET,简化书写，节省一点时间。

findOne在2.0版本不能直接传id，若要传id需要使用findById方法
用的是Girl girl = girlRepository.findById(id).get();来获取这一行记录

若想继续用findOne方法只能传一个对象过去，用下面的方法

import org.springframework.data.domain.Example;
@GetMapping(value = "/girls/{id}")
public Optional<Girl> girlFindOne(@PathVariable("id") Integer id)
{
   Girl girl = new Girl();
   girl.setId(id);
   Example<Girl> example = Example.of(girl);
   return girlRepository.findOne(example);
}
